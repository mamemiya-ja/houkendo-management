# 委託販売管理・精算請求システム

委託販売先ごとの納品、預け在庫、販売数、返品数、精算金額、A4帳票印刷を管理するローカル用アプリです。

## 起動方法

1. `index.html` をダブルクリックしてブラウザで開きます。
2. 左下の `自動保存先設定` を押し、台帳用のJSONファイルを保存する場所を選びます。
3. 右上の `サンプル投入` を押すと、試しに動かせるデータが入ります。
4. 実運用前に `設定` で自社名、住所、振込先、インボイス登録番号を入力してください。

Python、Node.js、データベースのインストールは不要です。

## 主要機能

- 取引先登録、編集、検索
- 商品登録、編集、検索、商品画像登録
- 委託納品登録と納品書A4印刷
- 精算登録、販売数入力、手数料・消費税・請求額の自動計算
- 委託販売精算書 兼 請求書のA4印刷
- 取引先別、商品別、長期滞留の在庫確認
- 取引先別、商品別の売上集計
- JSON形式でのデータ保存、読込

## データ保存について

ChromeまたはEdgeで開いた場合、初回に `自動保存先設定` で選んだJSONファイルへ、登録・編集のたびに自動保存されます。

次回起動時は、ブラウザに権限が残っていれば同じ台帳ファイルを自動で読み込みます。権限確認が出た場合は許可してください。

別PCへ移す場合や、別の台帳ファイルを使う場合は `台帳ファイル読込` からJSONを選びます。読み込んだファイルが以後の自動保存先になります。

念のため、左下の `手動バックアップ` から任意のタイミングでバックアップJSONをダウンロードできます。ブラウザのファイル自動保存機能に対応していない環境では、ブラウザ内の予備保存と手動バックアップを使ってください。

## 印刷・PDF保存

納品履歴や精算履歴の `印刷` ボタンを押すと、A4用の帳票が印刷対象になります。印刷画面で通常プリンターを選べば紙に印刷でき、`PDFに保存` を選べばPDFとして保存できます。

## 注意

ローカルファイル保存、OneDrive同期、Google Apps Script版、Supabaseクラウド同期版を用途に応じて選べます。入金管理、メール送信、本格的な複数事業者対応は将来拡張項目です。


## GitHub Pages + Supabaseクラウド同期版

PCとスマホで同期して使う本命構成として、GitHub Pages + Supabase + PWA に対応しました。

- 共同利用者ごとに別アカウントを使えます。
- 台帳は参加コードで共有します。
- 台帳データはSupabaseに保存され、参加メンバーだけが読み書きできます。
- スマホではホーム画面に追加して、アプリに近い形で使えます。

設定手順は [docs/SUPABASE_SETUP.md](docs/SUPABASE_SETUP.md) を参照してください。

## OneDrive同期について

Microsoft Graph APIを使い、OneDriveのアプリ専用フォルダに台帳JSONを保存できるようにしました。PCとスマホで同じMicrosoftアカウントを使えば、同じ台帳を読み書きできます。

利用には Microsoft Entra / Azure 側でアプリ登録を行い、アプリの `設定` 画面に Application (client) ID を入力する必要があります。詳しい手順は [docs/ONEDRIVE_GRAPH_SETUP.md](docs/ONEDRIVE_GRAPH_SETUP.md) を参照してください。

`file://` で直接開いた場合、OneDrive同期は利用できません。GitHub PagesなどのHTTPS URL、または `localhost` で開いてください。

## GitHub管理について

GitHubで管理する場合は、台帳JSONやバックアップJSONをコミットしないでください。運用メモは [docs/GITHUB_WORKFLOW.md](docs/GITHUB_WORKFLOW.md) にまとめています。

## Google Apps Script + Google Drive版

スマホとPCで同期して使うための簡易クラウド版を `gas-drive-app/` に追加しました。Google Apps ScriptのWebアプリとして公開し、台帳JSONをGoogle Driveに保存します。

Microsoft Entra / OneDrive Graph API の設定が不要なので、個人利用ではこちらの方が始めやすい構成です。詳しい手順は [gas-drive-app/README.md](gas-drive-app/README.md) を参照してください。

# 委託販売管理・精算請求システム

委託販売先ごとの納品、預け在庫、販売数、返品数、精算金額、A4帳票印刷を管理するクラウド台帳アプリです。データはSupabaseに保存し、GitHub Pagesから利用します。

## 起動方法

1. GitHub Pagesで公開されたアプリURLを開きます。
2. 左下の `ログイン/同期` からメールアドレスとパスワードでログインします。
3. 初回は `設定` で事業者名、住所、振込先、インボイス登録番号を入力してください。
4. `サンプル投入` を押すと、試しに動かせるデータが入ります。

保存先はSupabaseのクラウド台帳に一本化しています。

## 主要機能

- 取引先登録、編集、検索
- 商品登録、編集、検索、商品画像登録
- 委託納品登録と納品書A4印刷
- 精算登録、販売数入力、手数料・消費税・請求額の自動計算
- 委託販売精算書 兼 請求書のA4印刷
- 取引先別、商品別、長期滞留の在庫確認
- 取引先別、商品別の売上集計
- Supabaseへのクラウド保存と同期

## データ保存について

このアプリは1アカウントにつき1つの専用台帳をSupabaseに保存します。各アカウントは、それぞれ自分専用の台帳を使います。

ログイン後に登録・編集した内容はクラウドへ自動保存されます。明示的に同期したい場合は、左下の `ログイン/同期` から `クラウドから読込` または `クラウドへ保存` を押してください。

ブラウザ内には表示用の一時控えが残ることがありますが、運用上の正本はSupabase上の台帳です。

## 印刷・PDF保存

納品履歴や精算履歴の `印刷` ボタンを押すと、A4用の帳票が印刷対象になります。印刷画面で通常プリンターを選べば紙に印刷でき、`PDFに保存` を選べばPDFとして保存できます。

## セットアップ

Supabaseの設定手順は [docs/SUPABASE_SETUP.md](docs/SUPABASE_SETUP.md) を参照してください。

## 注意

`supabase-config.js` にはブラウザ用のProject URLとpublishable keyだけを置きます。Supabaseのservice role keyや個人用トークンは絶対にコミットしないでください。
